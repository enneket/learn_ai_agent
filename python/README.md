#### AI Agent 的学习代码
##### 教程
- [Hello-Agents](https://datawhalechina.github.io/hello-agents)
  
##### 运行
- 安装依赖
```bash
pip install -r requirements.txt
```
- 配置
  - 复制 `.env.example` 为 `.env`
  - 填写 OpenAI API 密钥等配置

- 运行
```bash
python -m agent_paradigm.re_act.run
python -m agent_paradigm.plan_and_slove.run
```
